## 1. 项目概述

### 1.1 项目背景

随着无人机技术的快速发展和应用场景的不断拓展，无人机租赁市场需求日益增长。本系统旨在构建一个专业的无人机租赁服务平台，为用户提供便捷的租赁服务，为商家提供高效的管理工具。

### 1.2 系统目标

- 提供用户友好的无人机浏览和租赁功能
- 实现完整的订单生命周期管理
- 支持用户资质认证和诚信管理
- 提供设备维护和故障处理流程
- 构建高效的后台管理系统

### 1.3 系统功能模块

```
翱翔无人机租赁系统
├── 用户端
│   ├── 用户认证（登录/注册）
│   ├── 设备浏览（列表/详情/搜索）
│   ├── 订单管理（下单/支付/收货/退租）
│   ├── 个人中心（资质/订单/评价）
│   └── 故障上报
│
└── 管理端
    ├── 控制台（数据统计/待办事项）
    ├── 用户管理（列表/状态/诚信）
    ├── 资质审核
    ├── 无人机管理（CRUD/库存）
    ├── 订单管理（发货/归还）
    ├── 故障审核
    ├── 维保管理
```

### 3.2 核心功能实现

#### 3.2.1 JWT身份认证

```java
// JwtUtil.java - JWT工具类
public class JwtUtil {
    private static final String SECRET = "drone-rental-secret-key";
    private static final long EXPIRE = 7 * 24 * 60 * 60 * 1000; // 7天

    // 生成Token
    public static String generateToken(Long userId, Integer role) {
        return JWT.create()
            .withClaim("userId", userId)
            .withClaim("role", role)
            .withExpiresAt(new Date(System.currentTimeMillis() + EXPIRE))
            .sign(Algorithm.HMAC256(SECRET));
    }

    // 验证Token
    public static DecodedJWT verifyToken(String token) {
        return JWT.require(Algorithm.HMAC256(SECRET))
            .build()
            .verify(token);
    }
}
```

#### 3.2.4 订单创建流程

```java
// OrderServiceImpl.java
@Transactional(rollbackFor = Exception.class)
public OrderVO createOrder(OrderCreateDTO dto) {
    Long userId = UserContext.getCurrentUserId();

    // 1. 检查用户状态
    userService.checkUserOperable(userId);

    // 2. 检查用户资质
    qualificationService.checkQualificationValid(userId);

    // 3. 检查无人机是否可租赁
    droneService.checkDroneRentable(dto.getDroneId());

    // 4. 计算租赁天数和金额
    Drone drone = droneService.getById(dto.getDroneId());
    long days = ChronoUnit.DAYS.between(dto.getStartDate(), dto.getEndDate()) + 1;
    BigDecimal totalAmount = drone.getPricePerDay().multiply(BigDecimal.valueOf(days));

    // 5. 创建订单
    RentalOrder order = new RentalOrder();
    order.setOrderNo(generateOrderNo());
    order.setUserId(userId);
    order.setDroneId(dto.getDroneId());
    // ... 设置其他字段
    this.save(order);

    // 6. 减少库存
    droneService.decreaseStock(dto.getDroneId(), 1, order.getId());

    // 7. 创建支付记录
    paymentService.createPayment(order);

    return convertToVO(order);
}
```

#### 4.2.1 Axios请求封装

```javascript
// api/request.js
import axios from 'axios'
import { ElMessage } from 'element-plus'
import { useAuthStore } from '@/stores/auth'

const service = axios.create({
  baseURL: '/api',
  timeout: 15000
})

// 请求拦截器
service.interceptors.request.use(
  config => {
    const authStore = useAuthStore()
    if (authStore.token) {
      config.headers['Authorization'] = `Bearer ${authStore.token}`
    }
    return config
  },
  error => Promise.reject(error)
)

// 响应拦截器
service.interceptors.response.use(
  response => {
    const res = response.data
    if (res.code !== 200) {
      ElMessage.error(res.message || '请求失败')
      if (res.code === 401) {
        const authStore = useAuthStore()
        authStore.logout()
        window.location.href = '/login'
      }
      return Promise.reject(new Error(res.message))
    }
    return res
  },
  error => {
    ElMessage.error(error.message || '网络错误')
    return Promise.reject(error)
  }
)

export const get = (url, params) => service.get(url, { params })
export const post = (url, data) => service.post(url, data)
export const put = (url, data) => service.put(url, data)
export const del = (url) => service.delete(url)
```

#### 4.2.2 状态管理（Pinia）

```javascript
// stores/auth.js
import { defineStore } from 'pinia'
import { ref, computed } from 'vue'
import { login, getCurrentUser } from '@/api/auth'

export const useAuthStore = defineStore('auth', () => {
  const token = ref(localStorage.getItem('token') || '')
  const user = ref(null)
  const isAdmin = ref(localStorage.getItem('isAdmin') === 'true')

  const isLoggedIn = computed(() => !!token.value)
  const isVerified = computed(() => user.value?.status === 1)

  const setToken = (newToken, admin = false) => {
    token.value = newToken
    isAdmin.value = admin
    localStorage.setItem('token', newToken)
    localStorage.setItem('isAdmin', String(admin))
  }

  const userLogin = async (credentials) => {
    const res = await login(credentials)
    const { token: newToken, ...userInfo } = res.data
    setToken(newToken, false)
    user.value = userInfo
    return res
  }

  const fetchUserInfo = async () => {
    if (!token.value) return null
    const res = await getCurrentUser()
    user.value = res.data
    return user.value
  }

  const logout = () => {
    token.value = ''
    user.value = null
    localStorage.removeItem('token')
    localStorage.removeItem('isAdmin')
  }

  return {
    token, user, isAdmin, isLoggedIn, isVerified,
    setToken, userLogin, fetchUserInfo, logout
  }
})
```

```javascript
      { path: 'users', name: 'UserManagement', component: () => import('@/views/admin/UserManagement.vue') },
      { path: 'drones', name: 'DroneManagement', component: () => import('@/views/admin/DroneManagement.vue') },
      { path: 'orders', name: 'OrderManagement', component: () => import('@/views/admin/OrderManagement.vue') }
    ]
  },
  { path: '/login', name: 'Login', component: () => import('@/views/auth/Login.vue') },
  { path: '/admin/login', name: 'AdminLogin', component: () => import('@/views/auth/AdminLogin.vue') }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

// 路由守卫
router.beforeEach((to, from, next) => {
  const authStore = useAuthStore()

  if (to.meta.requiresAuth && !authStore.isLoggedIn) {
    next('/login')
  } else if (to.meta.requiresAdmin && !authStore.isAdmin) {
    next('/admin/login')
  } else {
    next()
  }
})

export default router
```

## 7. 安全设计

### 7.1 身份认证

- 使用JWT Token进行身份认证
- Token有效期7天
- 敏感操作需要重新验证

### 7.2 权限控制

```java
// AdminInterceptor.java - 管理员权限拦截
@Override
public boolean preHandle(HttpServletRequest request, ...) {
    Integer role = UserContext.getCurrentUserRole();
    if (role == null || role != 1) {
        throw new BusinessException(ResultCode.FORBIDDEN);
    }
    return true;
}
```

### 7.3 数据安全

- 密码使用MD5加密存储
- 敏感信息脱敏展示（手机号、身份证号）
- SQL注入防护（MyBatis Plus参数化查询）
- XSS防护（前端输入过滤）

### 7.4 接口安全

- CORS跨域配置
- 请求频率限制（可扩展）
- 参数校验（@Validated）

## 8. 部署方案

### 8.1 环境要求

| 组件 | 版本要求 |
|------|----------|
| JDK | 17+ |
| MySQL | 8.0+ |
| Node.js | 16+ |
| Nginx | 1.20+ |

### 8.2 后端部署

```bash
# 1. 打包
mvn clean package -DskipTests

# 2. 运行
java -jar drone-rental-1.0.0.jar \
  --spring.profiles.active=prod \
  --server.port=8080
```

### 8.3 前端部署

```bash
# 1. 安装依赖
npm install

# 2. 构建
npm run build

# 3. 部署到Nginx
cp -r dist/* /usr/share/nginx/html/
```

### 8.4 Nginx配置

```nginx
server {
    listen 80;
    server_name localhost;

    # 前端静态文件
    location / {
        root /usr/share/nginx/html;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # API代理
    location /api/ {
        proxy_pass http://127.0.0.1:8080/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # 上传文件
    location /uploads/ {
        alias /data/uploads/;
    }
}
```

### 8.5 数据库初始化

```bash
mysql -u root -p < sql/init.sql
```

### B. 订单状态流转

```
0(待支付) ──支付──→ 1(待发货) ──发货──→ 2(待收货) ──收货──→ 3(租赁中) ──归还──→ 4(已归还)
    │                    │
    └──取消──→ 5(已取消)  └──退款──→ 6(已退款)
```
