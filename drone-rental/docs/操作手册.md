## 1. 系统概述

### 1.1 系统简介

翱翔无人机租赁系统是一个专业的无人机租赁服务平台，为用户提供各类无人机的在线租赁服务。系统分为用户端和管理端两个部分，用户可以浏览、租赁无人机，管理员可以管理设备、订单和用户。

### 1.2 系统角色

| 角色 | 说明 | 访问入口 |
|------|------|----------|
| 普通用户 | 可以浏览设备、租赁无人机、管理订单 | 前台首页 |
| 系统管理员 | 管理设备、订单、用户、审核等 | 后台管理 |

### 1.3 浏览器要求

- Chrome 80+（推荐）
- Firefox 75+
- Safari 13+
- Edge 80+

## 2. 用户端操作指南

### 2.1 用户注册与登录

#### 2.1.1 用户注册

1. 点击首页右上角「注册」按钮
2. 填写注册信息：
   - 用户名（必填，唯一）
   - 密码（必填，6位以上）
   - 确认密码
   - 手机号（必填）
3. 点击「注册」按钮完成注册
4. 注册成功后自动登录并跳转首页

#### 2.1.2 用户登录

1. 点击首页右上角「登录」按钮
2. 输入用户名和密码
3. 点击「登录」按钮

### 2.2 实名认证（飞行资质）

> **重要提示**：租赁无人机前必须完成实名认证

#### 2.2.1 提交认证

1. 登录后点击右上角头像，进入「个人中心」
2. 在「飞行资质」区域点击「提交认证」
3. 填写资质信息：
   - 证件类型（如：CAAC无人机驾驶员执照）
   - 证件号码
   - 有效期起止日期
   - 上传资质证书图片
4. 点击「提交」等待管理员审核

#### 2.2.2 认证状态说明

| 状态 | 说明 |
|------|------|
| 待审核 | 已提交，等待管理员审核 |
| 已通过 | 审核通过，可以租赁无人机 |
| 已拒绝 | 审核未通过，请查看拒绝原因并重新提交 |

### 2.3 浏览与搜索设备

#### 2.3.1 设备列表

1. 点击导航栏「设备浏览」进入设备列表页
2. 可按以下条件筛选：
   - 无人机类型（航拍/测绘/农业/巡检）
   - 品牌
   - 价格区间
3. 点击设备卡片查看详情

#### 2.3.2 设备详情

设备详情页展示：
- 设备图片
- 基本信息（品牌、型号、类型）
- 技术参数（续航时间、最大速度、载重等）
- 租赁价格（元/天）
- 库存数量
- 用户评价

### 2.4 租赁下单

#### 2.4.1 创建订单

1. 在设备详情页选择租赁日期范围
2. 系统自动计算租赁天数和费用
3. 确认无误后点击「立即租赁」
4. 跳转到支付页面

#### 2.4.2 订单支付

1. 确认订单信息
2. 选择支付方式（支付宝/微信/余额）
3. 点击「确认支付」
4. 支付成功后订单状态变为「待发货」

### 2.5 订单管理

#### 2.5.1 查看订单

1. 点击导航栏「我的订单」或在个人中心查看
2. 可按状态筛选订单

#### 2.5.2 订单状态流程

```
待支付 → 待发货 → 待收货 → 租赁中 → 已归还
   ↓
 已取消
```

| 状态 | 说明 | 可执行操作 |
|------|------|------------|
| 待支付 | 订单已创建，等待支付 | 支付、取消订单 |
| 待发货 | 已支付，等待商家发货 | - |
| 待收货 | 商家已发货 | 确认收货 |
| 租赁中 | 用户已收货，正在使用 | 申请退租 |
| 已归还 | 设备已归还 | 评价 |
| 已取消 | 用户取消订单 | - |

#### 2.5.3 确认收货

1. 收到设备后，在订单详情点击「确认收货」
2. 订单状态变为「租赁中」

#### 2.5.4 申请退租

1. 在「租赁中」的订单点击「申请退租」
2. 按商家要求寄回设备
3. 商家确认收货后订单完成

### 2.6 故障上报

如在使用过程中发现设备故障：

1. 进入相关订单详情
2. 点击「故障上报」
3. 填写故障信息：
   - 故障类型
   - 故障描述
   - 上传故障图片
4. 提交后等待管理员审核处理

### 2.7 评价管理

订单完成后可以评价：

1. 在「已归还」订单点击「评价」
2. 选择评分（1-5星）
3. 填写评价内容
4. 可上传图片（可选）
5. 提交评价

### 2.8 个人中心

#### 2.8.1 基本信息管理

- 修改昵称、邮箱
- 修改密码
- 查看诚信状态

#### 2.8.2 诚信状态说明

| 状态 | 说明 |
|------|------|
| 诚信良好 | 正常状态，可以正常租赁 |
| 诚信受限 | 存在违规行为，部分功能受限 |

## 3. 管理端操作指南

### 3.1 管理员登录

1. 访问管理后台登录页面
2. 输入管理员账号和密码
3. 默认管理员账号：admin / admin123

### 3.2 控制台（Dashboard）

登录后进入控制台，展示系统概览：
- 今日订单数
- 待处理任务数
- 本月收入
- 用户总数
- 订单趋势图表
- 待办事项列表

### 3.3 用户管理

#### 3.3.1 用户列表

1. 点击左侧菜单「用户管理」
2. 可搜索用户（昵称/手机号）
3. 可按状态筛选

#### 3.3.2 用户操作

| 操作 | 说明 |
|------|------|
| 查看详情 | 查看用户详细信息 |
| 启用/禁用 | 禁用后用户无法登录 |
| 诚信管理 | 标记/恢复诚信状态 |
| 重置密码 | 重置为默认密码 |

### 3.4 资质审核

#### 3.4.1 审核列表

1. 点击左侧菜单「资质审核」
2. 查看待审核的用户资质申请
3. 统计卡片显示：待审核、今日通过、今日拒绝、累计审核

#### 3.4.2 审核操作

1. 点击「详情」查看资质信息和证书图片
2. 审核通过：点击「通过」
3. 审核拒绝：点击「拒绝」并填写拒绝原因

### 3.5 无人机管理

#### 3.5.1 设备列表

1. 点击左侧菜单「无人机管理」
2. 查看所有无人机设备
3. 可按状态、类型筛选

#### 3.5.2 添加设备

1. 点击「添加设备」按钮
2. 填写设备信息：
   - 基本信息（型号、品牌、类型）
   - 技术参数（续航、速度、载重等）
   - 租赁信息（价格、库存）
   - 上传设备图片
3. 点击「保存」

#### 3.5.3 设备操作

| 操作 | 说明 |
|------|------|
| 编辑 | 修改设备信息 |
| 上架/下架 | 控制设备是否在前台展示 |
| 调整库存 | 增加或减少库存数量 |
| 删除 | 删除设备（逻辑删除） |

### 3.6 订单管理

#### 3.6.1 订单列表

1. 点击左侧菜单「订单管理」
2. 可按订单号、用户、状态、日期筛选

#### 3.6.2 订单操作

| 订单状态 | 可执行操作 |
|----------|------------|
| 待发货 | 发货（填写快递信息） |
| 租赁中 | 确认归还 |

#### 3.6.3 发货操作

1. 在「待发货」订单点击「发货」
2. 选择快递公司
3. 填写快递单号
4. 点击「确认发货」

#### 3.6.4 确认归还

1. 用户寄回设备后，点击「确认归还」
2. 检查设备状态
3. 确认后订单完成，库存自动恢复

### 3.7 故障报修审核

#### 3.7.1 故障列表

1. 点击左侧菜单「故障报修审核」
2. 查看用户提交的故障报告
3. 可按状态筛选

#### 3.7.2 审核故障

1. 点击「详情」查看故障信息和图片
2. 确认故障：点击「确认」并选择是否创建维修工单
3. 非故障：点击「驳回」并填写原因

### 3.8 维保管理

#### 3.8.1 维修工单列表

1. 点击左侧菜单「维保管理」
2. 查看所有维修工单
3. 可按状态筛选（待维修/维修中/已完成）

#### 3.8.2 处理维修工单

1. 点击「处理」进入工单详情
2. 填写维修信息：
   - 维修类型
   - 预估费用/天数
   - 指派维修人员
3. 更新维修进度
4. 完成维修后点击「完成」

### 3.9 评价管理

#### 3.9.1 评价列表

1. 点击左侧菜单「评价管理」
2. 查看所有用户评价
3. 可按状态筛选

#### 3.9.2 评价操作

| 操作 | 说明 |
|------|------|
| 回复 | 对用户评价进行回复 |
| 屏蔽 | 屏蔽不当评价（前台不展示） |
| 恢复 | 恢复已屏蔽的评价 |

## 4. 常见问题

### Q1: 忘记密码怎么办？

**用户端**：联系管理员重置密码
**管理端**：使用超级管理员账号重置

### Q2: 为什么无法租赁设备？

请检查：
1. 是否已登录
2. 是否已完成实名认证且审核通过
3. 诚信状态是否正常
4. 设备是否有库存

### Q3: 订单支付失败怎么办？

1. 检查网络连接
2. 确认支付账户余额充足
3. 重新点击支付
4. 如仍失败，请联系客服

### Q4: 设备出现故障怎么办？

1. 立即停止使用设备
2. 在系统中提交故障上报
3. 详细描述故障情况并上传图片
4. 等待管理员审核处理

### Q5: 如何申请退款？

已支付但未发货的订单：
1. 联系管理员申请退款
2. 管理员审核后进行退款操作
