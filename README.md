# 基于AI+Agent+Rag无人机设备器材租赁系统

#### 介绍
该项目是无人机的租赁系统，也可以是其他的设备、器材租赁，技术栈：Java 21 + Spring Boot 3.5.13 + Vue 3 + Spring AI + MCP + AI Memory，其他的技术栈Dify/Coze、 Python LangChain/LangGraph Agent 。项目是开源的，免费学习的，star支持一下，不断更新中。
